🔗 CodeAlpha_HangmanGame

🔗 Project overview

This project is a Python-based Hangman Game where players guess a randomly selected word one letter at a time. The game provides six attempts to reveal the hidden word, displaying correct guesses and tracking remaining attempts.
It is designed to demonstrate fundamental Python concepts through an interactive and engaging word-guessing experience.

🔗 Features

. Random word selection
. Menu-driven interface
. Letter-by-letter guessing
. Hidden word display using underscores
. 6 attempts to guess the word
.Correct letter updates
. Remaining attempts counter
.Win/Lose result display
. Simple and user-friendly gameplay

🔗 Concepts used

. Random
. While loop
. if-else
. Strings
. Lists
